# 🍔 Sistema de Pedidos via WhatsApp

Um catálogo digital interativo e responsivo (Mobile-First) desenvolvido com React. Este sistema permite que clientes naveguem pelos produtos, montem seus pedidos (incluindo pizzas personalizadas) e enviem tudo automaticamente para o WhatsApp do estabelecimento, com os valores e descrições formatados.

## ✨ Funcionalidades

- **Catálogo Dinâmico:** Listagem de produtos com imagens, preços e descrições.
- **Filtro por Categorias:** Navegação fácil entre "Lanches", "Pizzas", "Bebidas", etc.
- **Carrinho de Compras:** Adição, remoção e alteração de quantidade de itens em tempo real.
- **🍕 Função "Meia a Meia":** Modal exclusivo para montar pizzas de dois sabores diferentes, aplicando automaticamente a regra de negócio de cobrar pelo sabor mais caro.
- **Integração com WhatsApp:** Gera um link `wa.me` com o resumo completo do pedido, valor total e observações do cliente.
- **Design Responsivo:** Interface construída com Tailwind CSS para funcionar perfeitamente em celulares, tablets e desktops.

## 🚀 Tecnologias Utilizadas

- **[React](https://reactjs.org/)** (com Hooks: `useState`)
- **[Vite](https://vitejs.dev/)** (Build tool super rápido)
- **[Tailwind CSS v4](https://tailwindcss.com/)** (Estilização utilitária)
- **JavaScript (ES6+)**
